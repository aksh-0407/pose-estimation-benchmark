# benchmarks/

Benchmark output, split by what's safe to commit:

| Subdir | Committed? | Contents |
| ------ | ---------- | -------- |
| `runs/<run_id>/` | ✅ | Compact, immutable run evidence: manifests + metrics JSON + summary HTML. |
| `artifacts/<run_id>/` | ❌ local | Raw per-instance predictions and per-image logs (too big for Git; used for debug/resume). |
| `reports/` | ❌ derived | HTML reports generated by `benchmark.py report`; CI publishes them. |
| `datasets/` | ❌ local | Optional benchmark-local dataset assets. |

Run ids are unique and timestamped, so two people committing two runs never conflict.
Only `.gitkeep` placeholders, `README.md`, and `runs/<run_id>/` evidence are tracked.

Details: [docs/results-format.md](../docs/results-format.md) and
[docs/collaboration.md](../docs/collaboration.md).
