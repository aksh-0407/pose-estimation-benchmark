# models/

The canonical local store for model metadata and weights. One directory per model:

```text
models/<model_id>/
  model.yaml     # generated metadata (tracked)
  README.md      # generated description (tracked)
  weights/       # checkpoints (LOCAL, ignored)
  configs/       # model-specific config
  checksums/     # hashes of local weights (LOCAL, ignored)
```

`model.yaml`, `README.md`, and `checksums/` are generated by
[`sync_model_store.py`](../scripts/sync_model_store.py). Only metadata, configs, and
`.gitkeep` placeholders are tracked — weights and checksums stay local.

`external/` is reserved for upstream source clones; symlink from an upstream `ckpts/`
folder back into `weights/` if upstream code hard-codes a path.

See [docs/models.md](../docs/models.md) and
[docs/configuration.md](../docs/configuration.md).
